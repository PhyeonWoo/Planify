# Planify - SaaS 프로젝트 관리 플랫폼

[![Java](https://img.shields.io/badge/Java-21%2B-orange)](https://www.java.com/)
[![Spring Boot](https://img.shields.io/badge/Spring%20Boot-4.x-brightgreen)](https://spring.io/projects/spring-boot)

팀 협업을 위한 **프로젝트 관리 SaaS 플랫폼** 입니다.
\
\
\
\
🎯 **프로젝트 개요**

Planify는 팀의 프로젝트와 할일을 효율적으로 관리해주는 프로젝트입니다.
다중 워크스페이스 지원, 실시간 알림, 권한 관리, 댓글 기능 등을 통해 팀 협업을 극대화합니다.

**주요 특징**:

✅ 다중 워크스페이스 지원\
✅ 프로젝트 & 할일 계층적 관리 (서브태스크)\
✅ 실시간 알림 시스템\
✅ 권한 기반 접근 제어 \
✅ JWT 기반 인증\
✅ 대댓글 지원 댓글 시스템\
✅ 할일 상태/우선순위 관리\
✅ 사용자 맞춤 알림 설정
\
\
\
**Backend**
	•	Java
	•	Spring Boot
	•	MyBatis (또는 JPA 기반 가능성)

**Database**
	•	MySQL
  

\
\
1️⃣ **Workspace (워크스페이스)** - 팀 공간 관리

**개념**:\
워크스페이스는 팀이 협업하는 단위입니다. 개인 워크스페이스 또는 팀 워크스페이스를 생성할 수 있습니다.

**주요 기능**:

✅ 워크스페이스 생성/수정/삭제\
✅ 멤버 초대\
✅ 역할 기반 권한 관리 \
✅ 멤버 목록 조회 및 권한 변경

\
\
2️⃣ **Project (프로젝트)** - 프로젝트 관리

**개념**:\
워크스페이스 내의 프로젝트 단위입니다. 각 프로젝트는 독립적인 할일 관리 공간입니다.
주요 기능:

**주요 기능**:

✅ 프로젝트 생성/수정/삭제\
✅ 프로젝트별 멤버 관리\
✅ 상태 관리

\
\
3️⃣ **Task (할일/태스크)**

**개념**:\
프로젝트 내의 개별 작업 단위입니다. 계층적 구조(서브태스크)를 지원하며, 상태, 우선순위, 담당자를 관리합니다.
주요 기능:

**주요 기능**:

✅ 할일 생성/수정/삭제\
✅ 서브태스크\
✅ 상태 관리\
✅ 우선순위 설정\
✅ 담당자 할당\
✅ 드래그드롭 정렬\
✅ 댓글 및 대댓글\
✅ 내가 생성한 할일 / 내게 할당된 할일 조회

\
\
4️⃣ **Notification (알림)**

**개념**:\
할일 변경, 댓글, 초대 등의 실시간 알림 시스템입니다. 사용자 맞춤 알림 설정을 지원합니다.

**알림 타입**:\
TASK_DUE: 할일 마감일 알림\
TASK_ASSIGNED: 할일 할당\
COMMENT: 댓글 달림\
INVITE: 워크스페이스/프로젝트 초대

**주요 기능**:

✅ 알림 생성 \
✅ 알림 목록 조회\
✅ 미읽은 알림 조회\
✅ 읽음 처리 \
✅ 알림 설정 \
✅ 알림 삭제
